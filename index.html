Welcome to the Mining Project

A comprehensive workflow for processing and analyzing large datasets.

Overview

This project aims to process and analyze large datasets in multiple phases. Each phase involves reading a large file, processing the data, and generating new files for the next step. The entire workflow is designed to be modular and easy to follow.

Instructions

To run the entire project, follow these steps:

  1. Open the 1_download_data.ipynb notebook and run all cells to download the initial dataset.
  2. Open the 2_phase_one_processing.ipynb notebook and run all cells to process the data in the first phase.
  3. Open the 3_phase_two_processing.ipynb notebook and run all cells to process the data in the second phase.
  4. Open the 4_final_processing.ipynb notebook and run all cells to complete the final phase of data processing.

Alternatively, you can run the master_notebook.ipynb to execute all phases sequentially.
